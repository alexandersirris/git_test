Let's talk about the 3 different types of git commands.

  • Git Add will place a file w/ modified changes to the staging area. Files and changes at this point are not committed from the local repo to the remote repo.
  • Git Commit will commit the files and changes with a log. This records changes locally.
  • Git Push will push changes to the remote repo
